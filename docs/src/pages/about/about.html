Acerca de mí

Hola, soy Christopher Jiménez

Soy desarrollador web de lado del frontend. Contruyo sitios web dinamicos desde cero con estructuras solidas y escalables mediante Atomic Design, en el HTML y CSS implemento BEM agregando claridad en las secciones de la pagina y reutilización de estilos con pre-procesadores como SASS, tambien implemento su framework de Tailwind. Mi lenguaje de programación principal es JavaScript para hacer dinamico y escalable el sitio, uso de progamacion funcional y orientada a objetos, utilizo React para modularizar el proyecto con componentes y optimizarlo. Puedo crear prototipos base del proyecto con herramientas de diseño como Figma. Mi enfoque es darle al usuario una experiencia agradable, fluida e intuitiva en el sitio web.
