Acerca de mí

Hola, me llamo Christopher Jiménez

Mi perfil profesional es como desarrollador web front-end con enfoque en JavaScript. Mi formación base inició en la escuela de Desarrollo Web de Platzi y en paralelo de manera autodidacta. Puedo incorporarme a diferentes tipos de proyecto y brindar valor agregado, solucionando errores aplicando testeo, optimizando recursos y mejorando interactiva como visual el sitio web mediante bases de UX y UI, cuento con conocimientos sobre equipos de trabajo con metodologías ágiles como SCRUM para la comunicación efectiva en equipo y entregas a tiempo.

Experiencia

Cuento con un año y medio de experiencia trabajando en proyectos personales. He desarrollado diferentes sitios web dinámicos desde cero con estructuras sólidas y escalables mediante Atomic Design, en el HTML y CSS regularmente implemento BEM para tener una estructura clara de la página y sus estilos, me apoyo en preprocesadores como Sass para escalar los estilos y también hago uso del framework de Tailwind CSS. Me puedo adaptar a otros lenguajes de programación como TypeScript, Python entre otros gracias a las bases sólidas que tengo con JavaScript, trabajo la lógica con programación orientada a objetos y programación funcional, utilizo React para modularizar y reutilizar componentes haciendo más óptimo el proyecto. También me apoyo de herramientas como Figma para crear wireframes de alta fidelidad y tener un diseño base del proyecto. Mi enfoque es darle al cliente una experiencia agradable, fluida e intuitiva en el sitio web.
